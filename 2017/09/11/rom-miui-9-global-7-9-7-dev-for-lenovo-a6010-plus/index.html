MIUI 9 Global 7.9.7 for Lenovo A6010/Plus

Features:

  1. Rom name: MIUI 9
  2. Version: 7.9.7 Global
  3. Android: 5.1.1
  4. Ported by Phan Thanh
  5. Rooted

Screenshots: Link Download:

  1. Rom MIUI 9 beta 7.9.7 : here
  2. Gapps 5.1.1: here
  3. Browser: here
  4. Root.zip: here

Install (Hướng dẫn cài đặt):

  1. Download cả 3 link ở trên về bỏ vào thẻ nhớ (Để chỗ nào dễ nhớ để còn lấy ra dùng)
  2. Vào TWRP
  3. Wipe Dalvik, Data, Cache, System
  4. Vào Mount -> Bỏ tick ở System và Data
  5. Vào Install chọn file *miui9Global_7.9.7_5.1v1.zip *và cài đặt như mọi rom khác. Cái Root.zip vào rom mà root đc rồi thì không cần dùng nữa.
  6. Reboot -> đợi cho máy boot
  7. Sau khi đã boot lên. Kiểm tra xem có CHplay không? Nếu không thì cài đặt cái file Google Play Store.apk là sẽ có trở lại.

Chú ý(Notice): Rom này nó xoá Browser gốc rồi, nên cái thích browser gốc thì vào TWRP up cái patch Browser.zip ở link trên. Xem cách vào TWRP trên điện thoại tại đây
