Hướng dẫn thêm Customer Chat Facebook (LiveChat Facebook) vào website

PhanThanhGit
2018-05-13
Website Coding
facebook, website development

Bạn đang tìm kiếm một công cụ Live chat để hỗ trợ và trò chuyên với khách hàng cho website của bạn? Vậy thì Customer Chat Facebook là một công cụ mà bạn không nên bỏ qua. Với giao diện đơn giản, bắt mắt, hoạt động nhanh chóng, cài đặt nhanh chóng thông qua tài khoản fanpage Facebook sẽ là sự lựa chọn hoàn hảo cho bạn. Customer Chat Facebook là plugin cho Facebook phát hành, giúp các webmaster có thể tạo khung live chat nhanh chóng, hỗ trợ khách hàng. Tuy chỉ là bản beta nhưng những thứ mà plugin này mang lại còn hơn cả mong đợi. Sau đây mình sẽ hướng dẫn các bạn cách tích hợp Customer Chat Facebook vào website chỉ với vài bước đơn giản. Ưu điểm:

  • Hoàn toàn miễn phí (Không cần phải trả phí như một số Live chat khác)
  • Dễ cài đặt và vận hành.
  • Hỗ trợ người dùng nhanh chóng thông qua Facebook messenger.
  • Customer Chat Facebook hiển thị, hoạt đông tốt trên mọi loại trình duyệt trên PC, Mobile và Tablet.

Nhược điểm:

  • Người dùng cần có một tài khoản Facebook để đăng nhập.
  • Chưa thật sự có nhiều tùy biến.
  • Không thể theo dõi người dùng có đang truy cập Online hay không.

Các bước tích hợp Customer Chat Facebook vào website:

Yêu cầu cơ bản để tích hợp được Customer Chat Facebook vào website là:

  1. Bạn phải có một tài khoản Facebook và có một trang Fanpage.
  2. Bạn phải có 1 App Facebook để lấy ID cho phần source code của plugin có thể hoạt động.

Bước 1: Cấu hình plugin Customer Chat Facebook từ cài đặt của Fanpage Facebook.

Đầu tiên, bạn vào trang Fanpage Facebook, nhấn vào Cài đặt nằm ở góc phải trên màn hình. Tiếp theo là chọn Nền tảng Messenger nằm bên thanh menu bên trái. Nhấn chọn tiếp “Chuyển đển phần” Plugin Chat với khách hàng.

Tại khung Plugin chat với khách hàng chọn thiết lập:

Tại cửa sổ Setup Customer Chat chọn Next:

Tiếp theo bạn cần thiết lập vài thông số cho plugin:

  • Ngôn ngữ: Bạn có thể thiết lập ngôn ngữ tùy ý.
  • Lời chào: Để mặc định hoặc muốn tùy chỉnh bạn có thể nhấn vào “Thay đổi”.

Sau đó nhấn “Tiếp” để chuyển sang bước tiếp theo.

Tại bước 2 của thiết lập bạn có thể thay đổi màu khung chat, thời gian phản hồi và nhấn “Tiếp” để chuyển đến bước cuối cùng.

Ở bước cuối cùng của phần thiết lập bạn cần làm các việc sau:

  1. Thêm tên miền trang web của bạn vào và nhấn “Lưu”.
  2. Copy đoạn code mà Facebook cung cấp.
  3. Tick chọn “Tôi đã thêm mã vào trang web”. Nhấn “Hoàn tất” để hoàn thành việc thiết lập
