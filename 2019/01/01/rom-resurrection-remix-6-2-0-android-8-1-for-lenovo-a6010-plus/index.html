Rom Resurrection Remix 6.2.0 Android 8.1 for Lenovo A6010 plus

PhanThanhGit
2019-01-01
Rom OS
lenovo a6010, lenovo a6010 plus

Hế lô mấy cưng, hôm nay mình lại giới thiệu một rom 8.1 x32 cho Lenovo A6010 plus. Như tiêu đề, mình xin giới thiệu và hướng dẫn các bạn cài đặt Rom Resurrection Remix 6.2.0 Android 8.1 trên Lenovo A6010 plus. Lâu rồi không viết blog nên giờ không biết viết sao cho nó dài ra nữa. Thôi đi thẳng vào vấn đề luôn vậy.

Infor:

  1. Name: Resurrection Remix
  2. Version: 6.2.0
  3. Android version: 8.1
  4. Device: Lenovo A6010
  5. Type: Backup file
  6. Rooted
  7. No GAPPS
  8. Doldy
  9. ViPER

Review:

Links download:

  1. Rom: here
  2. Update Oreo: Update_Oreo.zip
  3. 3.10.108_32_Nitro_9+(H).zip
  4. Gapps: https://opengapps.org/ > ARM > 8.1 > pico > Download

Install:

  1. Upzip and copy Backup File Folder, Update_Oreo.zip and 3.10.108_32_Nitro_9+(H).zip into SD Card (Anh em lưu ý khi copy cái thư mục backup nhé, copy theo kiểu SdCard/rombackup/ mọi file backup anh em copy vào cái thư mục rombackup kia, chứ không copy thêm 1 bậc thư mục nữa nhé. Làm vậy nó mới nhận được file backup khi mình chọn file backup)
  2. Wipe all (data, system, cache, dalvik)
  3. Install Update_Oreo.zip
  4. Click Restore and select Backup File Folder, which you copied into SD Card at step 1.
  5. Install GAPPS
  6. Install 3.10.108_32_Nitro_9+(H).zip
  7. Reboot and enjoy.
