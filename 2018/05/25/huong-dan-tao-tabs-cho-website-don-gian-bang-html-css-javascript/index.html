Hướng dẫn tạo tabs cho website đơn giản bằng HTML, CSS, JavaScript

PhanThanhGit
2018-05-25
Website Coding
javascript

Tab là một form khá phổ biến và quen thuộc với nhiều website. Vậy họ đã tạo ra những kiểu tab đó bằng cách nào? Hôm nay, mình sẽ hướng dẫn cho các bạn cách đơn giản để tạo ra tabs với HTML, CSS và JavaScript. Để làm được bạn phải biết sơ qua về HTMl, CSS, và JS (javascript). Bước 1: Tạo giao diện cho tabs bằng HTML và CSS: Đầu tiên là code HTML. Gồm có 2 layout chính đó là layout cho tab (class: tab-menu) và layout cho nội dung tab (class: content-tab). Giờ thì style lại cho nó đẹp tí. Chèn code CSS sau vào :

* {
  font-family: 'Roboto', sans-serif;
}
.tab-menu {
  margin-top: 20px;
}
.tab {
  padding: 10px 20px;
  background: #304FFE;
  color: #fff;
  cursor: pointer;
  box-shadow: 0 3px 1px -2px rgba(0,0,0,.2), 0 2px 2px 0 rgba(0,0,0,.14), 0 1px 5px 0 rgba(0,0,0,.12);
}
.tab+.tab {
  margin-left: 10px;
}
.active {
  opacity: 0.5;
}
.content-tab-item {
  display: none;
}
.active-tab {
  display: inline-block;
}
