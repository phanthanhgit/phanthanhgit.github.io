Hướng dẫn tạo nút Back To Top cho website bằng Jquery 2018

PhanThanhGit
2018-06-18
Website Coding
javascript, jquery, thủ thuật website

Khi bạn lướt web, đôi khi bạn đọc phải một bài viết rất là dài. Lúc đọc xong bạn lại phải scroll chuột để quay lại đầu trang web để chuyển sang tin khác. Việc làm này tuy không tốn nhiều thời gian nhưng sẽ khiến chúng ta cảm thấy khó chịu khi phải lặp đi lặp lại chuyện đó nhiều lần. Vậy thì hôm nay mình xin hướng dẫn các bạn cách tạo nút Back To Top cho website của bạn một cách đơn giản bằng Jquery. Mình không dài dòng nữa, mình xin vào hướng dẫn luôn. Đầu tiên bạn cần thêm thư viện Jquery vào website. Copy đoạn code sau thả vào thẻ :

Tiếp theo, bạn thêm thư viện icon fontawesome này vào cho mình vì chúng ta sẽ dùng đến nó. Copy và dán vào thẻ :

Bây giờ, chúng ta cùng vào những giai đoạn chính để tạo ra nút Back To Top. Đầu tiên là chúng ta tạo layout và style cho nút Back To Top.

Tạo giao diện (stylesheet) cho cái nút đó:

#back-to-top {
cursor: pointer;
position: fixed;
bottom: 20px;
right: 20px;
background: #189EFF;
color: #fff;
padding: 15px;
}

Cuối cùng là viết hàm Jquery cho cái nút đó hoạt động:

$(document).ready(function(){
$(window).scroll(function(){
if ($(this).scrollTop() > 100) { //thực hiện lệnh điều kiện Khi lăn chuột xuống dưới hơn 100px
$(‘#back-to-top’).fadeIn(); //Xuất hiện nút
} else {
$(‘#back-to-top’).fadeOut(); //Ngược lại thì ẩn nút
}
});
$(‘#back-to-top’).click(function(){
$(“html, body”).animate({ scrollTop: 0 }, 600); //Animation giúp hoạt ảnh scroll ngược lên đầu trang sẽ mượt hơn
return false;
});
});

Bạn nên đặt đoạn code jquery này ở đầu trang hoặc cuối trang dưới

. Tuy nhiên mình khuyên là nên đặt ở đầu để khi load trang nó không bị hiển thị nút Back To Top lên trước mặc dù khi mới load ta chưa hề scroll chuột. Đây là thành quả. Chúc các bạn thành công.
