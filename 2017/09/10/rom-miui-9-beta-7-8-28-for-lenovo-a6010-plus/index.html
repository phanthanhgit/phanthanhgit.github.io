    *MIUI 9 beta 7.8.28 for Lenovo A6010/Plus *

    Features:

    1. Rom name: MIUI 9 beta
    2. Version: 7.8.28
    3. Android: 5.1.1
    4. Ported by Phan Thanh
    5. Deleted China App
    6. Added some app from MIUI 9 beta of Redmi 3
    7. No Gapps
    8. Rooted
    9. New launcher

    Source: here Screenshots: Link Download:

    1. Rom MIUI 9 beta 7.8.28 : here
    2. Link fix keyboard: here
    3. link fix mms: here
    4. Gapps 5.1.1: here
    5. MobileUncle Tools: here

    Install (Hướng dẫn cài đặt):

    1. Download cả 3 link ở trên về bỏ vào thẻ nhớ (Để chỗ nào dễ nhớ để còn lấy ra dùng)
    2. Vào TWRP
    3. Wipe Dalvik, Data, Cache, System
    4. Vào Mount -> Bỏ tick ở System và Data
    5. Vào Install chọn file developer_wt86047_7.8.28_v5.1.1.zip *và cài đặt như mọi rom khác. Tiếp theo là cái Parth *keyboard.zip và mms.zip
    6. Reboot -> đợi cho máy boot được vào Màn hình chính rồi mới làm bước 7 nhé
    7. Sau khi máy đã khởi động xong và vào được màn hình chính thì nhìn xem trên màn hình có ứng dụng MobileUncle Tools hay không. Nếu có thì bật nó lên. Nếu không thì cài đặt nó bằng file APK đã download ở link trên. Xem ảnh dưới để biết app đó như thế nào

    8. Mở MobleUncle Tools lên -> Cấp quyền root nếu nó yêu cầu -> Trượt xuống dưới tìm dòng Into Recovery Mode và Click vào -> OK. Cách này để bạn vào lai TWRP để cài Gapps cho rom 9. Sau khi đã vào lại được TWRP -> Install -> Chọn Gapps đã tải ở trên để cài đặt -> Chờ nó cài xong thì Reboot. 10. Finished.
