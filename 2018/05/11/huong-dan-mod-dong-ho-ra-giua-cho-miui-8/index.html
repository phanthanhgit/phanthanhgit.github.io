Hướng dẫn mod đồng hồ ra giữa cho MIUI 8

PhanThanhGit
2018-05-11
Mod Android
android, mod

Chỉ với vài thao tác đơn giản bạn có thể biến status bar của MIUI 8 trông giống ios với đồng hồ ra giữa, sóng wifi sang trái, …. Công cụ:

  1. apktool từ 2.1 trở lên
  2. Các file apk gồm: framework-res.apk, framework-ext-res.apk, miui.apk, MiuiSystemUI.apk
  3. Notepad++

Các bước thực hiện:

  1. Cài đặt framework cho apktool: Ở đây các bạn cài 3 file framework-res.apk, framework-ext-res.apk, miui.apk
  2. Decompile MiuiSystemUI.apk được thư mục MiuiSystemUI
  3. Tìm file theo đường dẫn res/layout/status_bar_simple.xml
  4. Thay toàn bộ code trong file status_bar_simple.xml bằng code sau:
