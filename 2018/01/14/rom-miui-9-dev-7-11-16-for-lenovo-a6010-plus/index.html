Cài đặt:

  1. Tải tất cả các file ở trên về thẻ SD
  2. Vào TWRP
  3. Wipe all (dalvik, system, cache, data)
  4. Vào Mount và bỏ tick system, data (untick system, data)
  5. Vào Install và chọn rom miui9 7.11.16.zip để cài đặt
  6. Rom chưa root nên có thể cài thêm UPDATE-SuperSU-v2.46.zip để root
  7. Reboot

Lưu ý: Nếu không nhìn thấy CHPlay trên màn hình thì cài đặt file *Google Play Store.apk *
