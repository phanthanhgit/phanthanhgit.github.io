How to fix wifi for Lenovo A6010 Plus

Hôm nay mình xin hướng dẫn các bạn cách fix wifi cho Lenovo A6010 Plus khi cài một số rom như ColorOs 2.1.0i, ColorOs 3.0,.v.v… Dụng cụ:

  1. Máy đã root
  2. Root Explorer.
  3. File dữ liệu cần dùng: download

Các bước thực hiện:

  1. Cài đặt Root Explorer, các bạn có thể tải từ ChPlay.
  2. Mở Root Explorer cấp quyền root cho Root Explorer.
  3. Giải nén file data vừa download ở trên.
  4. Vào thư mục vừa giải nén theo đướng dẫn speedsoftware/extract/system/etc/firmware/wlan/prima/
  5. Copy toàn bộ file trong thư mục prima vào máy với đường dẫn tương tự system/etc/firmware/wlan/prima/
  6. Set quyền 0777 cho các file vừa copy vào, có 2 file nhé.
  7. Reboot
  8. Lưu ý, nếu cảm thấy wifi đã bắt được từ 3 đến 4 vạch (cái này tùy theo wifi bạn đang dùng mạnh hay yếu nữa, tìm wifi thật mạnh test cho chắc) thì không làm bước tiếp theo
  9. Nếu wifi vẫn không bắt được từ 3 đến 4 vạch (cái này tùy theo wifi bạn đang dùng mạnh hay yếu nữa, tìm wifi thật mạnh test cho chắc) thì copy tương tự các thư mục còn lại trong thư mục mục vừa giải nén vào system/etc/ nếu vẫn chưa được thì copy luônsystem/lib/ vào nhé, nhưng set quyền khác tùy theo vị trí từng file nhé.
  10. Reboot and enjoy.

Các bạn có thể tham khảo video dưới đây. https://www.youtube.com/watch?v=ietDNLV-lJo Chúc các bạn thành công.
