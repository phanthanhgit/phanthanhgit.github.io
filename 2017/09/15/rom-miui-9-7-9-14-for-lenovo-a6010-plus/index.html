    MIUI 9 China 7.9.14 for Lenovo A6010/Plus

    Features:

    1. Rom name: MIUI 9
    2. Version: 7.9.14 dev CN
    3. Android: 5.1.1
    4. Ported by Phan Thanh
    5. Rooted

    Screenshots: Link Download:

    1. Rom MIUI 9 beta 7.9.14 dev CN: here
    2. Gapps 5.1.1: here
    3. MobileUncle Tools: here

    Install (Hướng dẫn cài đặt):

    1. Download cả 3 link ở trên về bỏ vào thẻ nhớ (Để chỗ nào dễ nhớ để còn lấy ra dùng)
    2. Vào TWRP
    3. Wipe Dalvik, Data, Cache, System
    4. Vào Mount -> Bỏ tick ở System và Data
    5. Vào Install chọn file developer_wt86047_7.9.14_v5.1.1.zip và cài đặt như mọi rom khác
    6. Reboot -> đợi cho máy boot lên
    7. Sau khi máy đã khởi động xong và vào được màn hình chính thì nhìn xem trên màn hình có ứng dụng MobileUncle Tools hay không. Nếu có thì bật nó lên. Nếu không thì cài đặt nó bằng file APK đã download ở link trên. Xem ảnh dưới để biết app đó như thế nào [![](https://phanthanhblog.com/wp-content/uploads/2017/09/Screenshot_2017-09-07-21-04-22-366_com.miui.home_-169x300-169x300.png)](https://phanthanhblog.com/wp-content/uploads/2017/09/Screenshot_2017-09-07-21-04-22-366_com.miui_.home_-169x300.png)_
    8. MIUI 9 beta 7.9.14 for Lenovo A6010[/caption] 8. Mở MobleUncle Tools lên -> Cấp quyền root nếu nó yêu cầu -> Trượt xuống dưới tìm dòng Into Recovery Mode và Click vào -> OK. Cách này để bạn vào lai TWRP để cài Gapps cho rom 9. Sau khi đã vào lại được TWRP -> Install -> Chọn Gapps đã tải ở trên để cài đặt -> Chờ nó cài xong thì Reboot. 10. Finished.
